How to Install and Use VS Code's Live Sever Extension

Installation Inscructions

  1. In Visual Studio Code, click the Extensions icon in the left column toolbar.
  2. In the search bar that appears, type Live Server.
  3. Select the Live Server extension. It should be that top choice.
  4. In the page for Live Server that comes up, click the green Install button.

Usage Instructions

  • Visual Studio Code must be open to a folder to use Live Server.
  • In VS Code's Explorer, right click (Windows) or Control+click (macOs) on an HTML file. From the context menu that appears, select Open with Live Server.
